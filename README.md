# Logistic Regression Project

## 📌 Overview
This project demonstrates the implementation of **Logistic Regression** using Python for classification tasks.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## 🚀 Features
- Data preprocessing
- Model training
- Prediction
- Model evaluation

## ▶️ How to Run
1. Open `Logistic_regression.ipynb`
2. Run all cells in Jupyter Notebook

## 📊 Output
The model is evaluated using:
- Accuracy Score
- Confusion Matrix
- Classification Report
